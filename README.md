#### 安装环境
- 访问`https://nodejs.org/zh-cn/`下载合适的版本, 安装完成后打开命令行(windows下是cmd)
- **进入该项目目录**, 输入以下命令(确保answerId已经填入index.js中)
```bash
npm i
node index.js
```
等待片刻, 完成后打开命令行中返回的网址即可
